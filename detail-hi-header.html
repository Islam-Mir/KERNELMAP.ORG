DESCRIZIONE:

I file header del kernel Linux sono file di intestazione (.h) che definiscono interfacce, strutture dati, costanti e macro utilizzati dal codice del kernel e dai moduli.

CONFRONTO:

Come le pagine di un libro che spiegano i termini tecnici usati nel testo principale: i file header definiscono il "vocabolario" che il kernel usa per comunicare con se stesso e con i moduli esterni.
